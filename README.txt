Правила по установке

1. Скачиваем проект
2. Генерируем .env
3. Заполняем данными
	- MySql доступы
	- База "websocket"
4. Настраиваем Pusher
	-Ключи доступны после регистрации

5. Генерируем ключи
6. Запускаем миграции
7. Запускаем очереди
8. Запускаем npm run dev(при необходимости)

Писалось под виндой поэтому для линукса в public нужна папка upload с правами на запись

--------------------------------------------------------------------------------

mysite/evenInit - страница с формой для создания новой статьи
	Создает форму. Записывает в бд. Инициирует старт события		

mysite/listen - страница для отрисовки форм

	Слушает событие создания форм. Отрисовывает формы.
